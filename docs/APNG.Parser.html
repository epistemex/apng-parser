Class: Parser

APNG. Parser

new Parser(input, callback [, onerror])

Parses a Animated PNG (APNG) into raw frames (images) which can be used for "manual" animation, frame extraction, analyze and optimization purposes.

The parsing is asynchronous and require a callback function. Callback for errors is optional.

Parameters:
Name Type Argument Description
input ArrayBuffer | TypedArray | String | Blob | File

URL to a APNG file, or an Blob/File object, or a ArrayBuffer/TypedArray holding a APNG file.
callback function

callback function invoked when all parsing and conversion is done. this represents the current instance.
onerror function <optional>

error callback.
